# 운동과 이중과제

운동 화면은 A/B/C/D 케어 유형에 따라 다른 운동 흐름을 제공한다. 1단계는 워밍업, 2단계는 음악 기반 운동, 3단계는 이중과제, 4단계는 쿨다운이다.

3단계 이중과제는 몸과 인지를 동시에 쓰는 운동이다. A유형은 과일/동물 단어를 듣고 걷기와 멈춤을 수행한다. B유형은 오른쪽/왼쪽 방향 순서를 기억하고 사이드스텝을 한다. C유형은 단어 글자 수만큼 무릎 펴기를 한다. D유형은 "커진다/작아진다" 같은 상상 놀이 명령에 맞춰 다리를 벌리고 모은다.

센서는 운동 중 계속 인식할 수 있다. 실제 서비스에서는 사용자가 직접 측정한 기준값 또는 캘리브레이션 값을 저장해 오른쪽/왼쪽, 걷기/멈춤, 반복 횟수, 다리 벌림/모음 같은 동작을 판정한다.

펭트는 센서 판정 결과에 따라 안내, 재시도, 칭찬을 제공한다. 사용자가 움직이지 않으면 "괜찮아요, 천천히 다시 해볼까요?"처럼 재시도 멘트를 하고, 성공하면 "좋아요, 방금 아주 잘하셨어요."처럼 응원한다.

# 음악과 TTS

BGM과 효과음은 `/static/audio`의 mp3 파일을 사용한다. 펭트 멘트는 별도 TTS로 재생하는 구조가 좋다. 음악 파일에서 목소리를 추출하기보다, BGM은 배경음악으로 두고 펭트 멘트를 별도 TTS 또는 mp3로 얹는 방식이 안정적이다.

펭트가 말할 때는 BGM 볼륨을 낮추고, 말이 끝나면 원래 볼륨으로 되돌린다. 사용자가 말하는 STT 구간에서는 펭트 TTS가 마이크에 들어가지 않도록 펭트 말하기 상태를 추적해 입력을 무시하거나 echo guard를 적용한다.
