# Legacy gait analysis scripts

이 폴더는 2026-07-22 최종 모델 확정 전에 사용한 실험용/비채택 분석 스크립트 보관소다.

현재 최종 모델 기준으로 유지할 주요 스크립트는 다음 파일이다.

- `analysis_scripts/retrain_acconly_clean.py`
- `analysis_scripts/compare_final_speed_or6_models.py`
- `analysis_scripts/build_75h_subwindow_median_iqr.py`
- `analysis_scripts/calibrate_waist_sensor_range_loss.py`

이 폴더의 스크립트는 pitch/gyro, bout-level, best10, OR4 등 후보 실험 재현용이며 현재 서비스 모델 기준 README에는 포함하지 않는다.
